# Sample output schema

Each entry in `gloves_sample.json` and `sutures_sample.json` is a
single product row from `output/products.db` with JSON columns deserialized.

See README.md → "Output schema" for the canonical field reference.

## Files
- `gloves_sample.json` — 5 representative products from "Dental Exam Gloves"
- `sutures_sample.json` — 5 representative products from "Sutures & surgical products"

## Selection
Samples are picked from the most-complete records (highest count of filled
optional fields) so the reader sees the schema in its filled-out form. See
the README for full per-field coverage statistics across the 873-product run.
